# Solutions to "Object-Oriented Programming in C++" by Robert Lafore

This repository contains my solutions to the programming exercises in Robert Lafore's "Object-Oriented Programming in C++".

## Structure

The solutions are organized by chapter:

*   `Chapter_02_Basics`
*   `Chapter_03_Loops`
*   `Chapter_04_Structures`
*   ...and so on.

Each chapter folder contains the solution files named `problem_1.cpp`, `problem_2.cpp`, etc.
